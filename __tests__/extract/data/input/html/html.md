# HTML

## Таблицы br, hr

| Заголовок 1                               | Заголовок 2  |
| ----------------------------------------- | ------------ |
| Текст на строке 1 <br> Текст на строке 2  | Текст        |
| Текст над линией <hr> Текст под линией    | Текст        |

## Мульти-строчные таблицы html blocks

#|
|| **Заголовок 1** | **Заголовок 2** ||
||

    текст внутри дива
</div>
| Обычный текст 2 ||
|#
